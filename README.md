# job-dealer

## Installation
To create services and launch in containers, run `start.sh`.
```bash
$ bash /start.sh
```

## Technologies
#### Backend
- Spring boot
- Spring Data JPA
- Spring Security
- Postgres
- Jackson
- Lombok
- Dozer
- Swagger 2
- Maven
- OpenJDK 8

#### Frontend
- Node
- Npm
- Vue/Vuex
- Webpack
- Bootstrap 4
- Javascript
- HTML/CSS
